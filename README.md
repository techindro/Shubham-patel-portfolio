# Shubham Patel - Full Stack Portfolio

A modern, full-stack portfolio website with authentication, contact management, and analytics.

## 🚀 Features

- **Full Stack Architecture**: Node.js/Express backend with MongoDB database
- **User Authentication**: JWT-based signup/login system
- **Contact Management**: Store and manage contact form submissions
- **Analytics Tracking**: Page views and visitor analytics
- **Newsletter System**: Email subscription management
- **Dynamic Content**: Portfolio data loaded from database
- **Responsive Design**: Mobile-first, modern UI
- **Admin Panel**: Manage messages and view analytics (admin only)

## 🛠️ Tech Stack

### Frontend
- HTML5, CSS3, JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Playfair Display, DM Sans, DM Mono)

### Backend
- Node.js with Express
- MongoDB with Mongoose
- JWT Authentication
- Bcrypt for password hashing
- Express Validator for input validation
- Rate limiting & security headers
